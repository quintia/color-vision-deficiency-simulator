# Color Vision Deficiency Simulator

macOS 用のアプリ Sim Daltonism のコードが Apache License 2.0 で公開されていたことから、Codex を用いてアルゴリズムを JavaScript に再現したスタンドアロンアプリを作成しました。画像はアップロードされず、ブラウザ内で処理します。

This is a standalone app that reproduces a color blindness simulation algorithm in JavaScript using Codex, based on the macOS app Sim Daltonism released under the Apache License 2.0. Images are processed locally in the browser.

## Repository

- Source repository: https://github.com/quintia/color-vision-deficiency-simulator
- Upstream reference (original app source): https://github.com/michelf/sim-daltonism
